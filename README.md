# Install_ROS
## Install ROS melodic
Execute the following commands:
```
$ cd ~/Install_ROS
$ ./install_melodic_ubuntu.sh
```
## Install ROS noetic
Execute the following commands:
```
$ cd ~/Install_ROS
$ ./install_noetic_ubuntu.sh
```
## Install ROS2 humble
Execute the following commands:
```
$ cd ~/Install_ROS
$ ./install_humble_ubuntu.sh
```
